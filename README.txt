Размещение в интернете (GitHub Pages):
1. Создай репозиторий на GitHub, залей содержимое папки (index.html, style.css, script.js, images/).
2. В настройках репозитория включи GitHub Pages (branch: main, folder: / (root)).
3. Подожди ~1 минуту и сайт будет доступен по адресу: https://<твой-логин>.github.io/<репозиторий>

Netlify (быстро и просто):
1. Зарегистрируйся на netlify.com и подключи репозиторий (GitHub / GitLab).
2. Выбери основной branch и нажми Deploy — сайт будет доступен через Netlify URL.
3. Для собственного домена — в настройках добавь домен.
